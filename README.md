# ChillEasy

ChillEasy is a smart cooling and environmental monitoring system designed to provide efficient temperature control, real-time monitoring, and an intuitive user experience. The project combines hardware, firmware, backend services, and a frontend application to create a complete end-to-end solution.

## Features

* Real-time temperature and environmental monitoring
* Smart cooling control and automation
* User-friendly web/mobile interface
* Backend APIs for device communication
* Firmware support for embedded hardware
* Data logging and analytics
* Scalable and modular architecture

## Installation

### Clone Repository

```bash
git clone https://github.com/SumitGyawali/ChillEasy.git
cd ChillEasy
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Firmware Setup

1. Open the firmware project in the appropriate IDE.
2. Configure device credentials.
3. Build and flash to the microcontroller.

## Usage

1. Start the backend server.
2. Launch the frontend application.
3. Connect the hardware device.
4. Monitor and control the system through the dashboard.
